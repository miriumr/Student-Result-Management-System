# 🎓 Student Result Management System (SRMS) Prototype

## 📌 Overview

The **Student Result Management System (SRMS) Prototype** is a mobile application developed using **MIT App Inventor**. It is designed to demonstrate a simple digital system for managing student records and results. The app helps in storing, updating, and viewing student data in an organized and user-friendly way.

---

## ✨ Features

* Add new student records
* Store student results
* View student details easily
* Simple and interactive UI
* Basic prototype for understanding system workflow

---

## 🛠️ Tools & Technologies

* MIT App Inventor
* SQLite / Local Storage (if used)
* Drag-and-drop UI design
* Block-based programming logic

---

## 📱 How It Works

1. Admin/user opens the app
2. Student data is entered through input forms
3. Data is stored locally
4. User can view or update records anytime

---

## 🎯 Purpose

This project is developed as a **prototype** to demonstrate the basic concept of a Student Result Management System. It focuses on core functionality and workflow rather than advanced features, serving as a foundation for future improvements.
